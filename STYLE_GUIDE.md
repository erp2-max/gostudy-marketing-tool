## 1. 品牌定位

### 核心身份
GoStudy 是專注於台灣留學市場的「海外名校申請策略顧問」，而非單純的資訊整合平台。

### 品牌主張
> 「不只是申請學校，而是透過**策略布局**與**價值重塑**，將學生的平凡經歷轉化為招生官眼中的亮點故事。」

### 信任背書
- 顧問團隊畢業自 Imperial College London 等世界頂尖名校
- 深耕台灣留學市場 10 年以上
- 超過 3,000 名成功學生案例

### 差異化定位
| GoStudy | 一般留學代辦 |
|--------|------------|
| 策略性價值重塑 | 表單填寫協助 |
| 客製化 SOP 故事撰寫 | 套版文書處理 |
| 深度選校分析（就業導向） | 排名導向選校 |
| 非典型背景專業轉化 | 僅接受條件優秀學生 |

## 2. 目標受眾

### Persona A｜頂尖學霸
- **特徵**：目標常春藤 (Ivy League)、英國 G5、澳洲 Go8
- **痛點**：背景高度同質化，不知如何在申請中脫穎而出
- **訴求**：突破競爭，讓招生官記住自己

### Persona B｜非典型背景
- **特徵**：GPA 2.5–2.8、非本科系轉考、缺乏 GRE/GMAT
- **痛點**：擔心條件不夠、被直接篩掉
- **訴求**：找到能接受並欣賞自己背景的學校與策略

### Persona C｜跨領域轉職者
- **特徵**：工作 3–8 年，欲透過留學升級職涯賽道
- **痛點**：不確定哪個國家/學位對自己最有效
- **訴求**：明確的 ROI（投資報酬率）與就業保障

### Persona D｜迷惘的高中生與家長
- **特徵**：剛考完學測、對選校排名感到焦慮
- **痛點**：不知道備案怎麼選、怕選錯路
- **訴求**：有人幫忙規劃清楚的升學路徑

## 4. 配色規則

### 主色系

```
品牌金／卡其色：#A68A56
深背景色：      #1A1A2E（深藍黑）
白色文字：      #F5F5F0
強調黃（關鍵字高亮）：#FFD700
低彩冷灰：      #8B9EA8
```

### 使用邏輯

- **`#A68A56`（品牌金）**：框線、標籤底色、LOGO 配色、分隔線
- **深色背景**：搭配白色文字，創造高對比專業感
- **強調黃**：圖卡內文的關鍵字 highlight、數字強調
- **冷灰**：副標、補充說明、不強調的資訊

### ❌ 禁止配色
- 紫色漸層（AI 濫用色，損害品牌辨識度）
- 純白底 + 黑字（過於普通，無品牌感）
- 高飽和彩色系（如螢光綠、鮮紅）

## 5. 排版規則

### 文字層級

```
層級 1（大標）：吸引注意力，4–8 字，最大字體
層級 2（副標）：痛點共鳴或具體數據，8–15 字
層級 3（內文）：策略說明，每段最多 3 句話
層級 4（CTA）：行動呼籲，1 句話，置底
```

### 排版規則一覽

| 規則 | 說明 |
|-----|------|
| 標題分隔符 | 統一使用全形「｜」，如「排名｜策略｜錄取」 |
| 標題字數 | 控制在 4–8 字內 |
| 每段句數 | 最多 3 句，結論優先 |
| Bullets 數量 | 每頁最多 4 項，輪播圖卡 6–8 張為上限 |
| 關鍵字強調 | 使用黃色底色或粗體，每段最多 2 處 |
| Emoji 位置 | 標題前後或行首，每行最多 1–2 個 |

## 6. 文案語氣

### 核心語氣關鍵字

> 自信、挑戰、洞察、驚嘆、數據導向、台灣在地化

### Emoji 使用規範

```
高能量：‼️ ❗️ 🔥
學術權威：🎓 📚 🏛️
確認正向：✅ 💯
秘密揭露：㊙️ 🔍
警示提醒：⚠️ 🚨
```

> **原則**：每則貼文 Emoji 總數控制在 5–10 個，避免視覺雜亂。

### 常用台灣在地化詞彙

| 詞彙 | 使用情境 |
|-----|---------|
| 鍍金 | 留學帶來的履歷升級效果 |
| 還在觀望 | 催促猶豫受眾採取行動 |
| 神隊友 | 形容 GoStudy 顧問的角色 |
| 避坑 | 幫助學生避免常見錯誤 |
| 荷包君 | 輕鬆討論費用話題 |
| 轉化策略 | 將弱點轉為優勢的申請方法 |
| 賽道修正 | 調整選校方向 |
| 逆襲 | 非典型背景成功案例敘事 |

### 語氣禁忌
- ❌ 過度謙遜（如「我們只是小顧問」）
- ❌ 模糊承諾（如「有機會申請到好學校」）
- ❌ 負面沉默（不討論 Study Gap 或低 GPA）

## 7. 標題公式

### 公式一｜衝突破題型
```
[A 優於 B 的指標]，[B 卻贏在 C]？！

範例：
- UCLA 排名更勝，薪水卻輸柏克萊？！
- QS 前 10 更名校，就業力卻輸排名 50？！
- 雇主最愛 Top10 竟沒牛津劍橋？
```

### 公式二｜逆襲結果型
```
[弱勢條件] 竟直升 [亮眼成果]！

範例：
- GPA 2.8 竟直升世界百大名校商科！
- 工程學分不足，也能進常春藤？
- 雅思卡關，卻錄取全澳 TOP1 墨爾本大學！
```

### 公式三｜權威數據型
```
[年份] [權威來源] [數據事件]

範例：
- 2026 QS 大學排名，英國名校大洗牌
- 英美名校直達車！AP/IB 學程的終極升學指南
- 教育部最新公告：這 5 所學校台灣雇主最買單
```

### 公式四｜急迫痛點型
```
[痛點事件] ＋ 暗示緊迫感

範例：
- 雅思暴漲 800 元！再不報名就晚了
- 90% 留學申請都出錯！你中了幾個？
- 申請截止倒數 30 天，你的 SOP 寫好了嗎？
```

### 公式五｜模板填空型
```
[條件 A] 逆襲 [名校 B]
[痛點 C] 的秘密武器
[事件 D] 竟是 [誤區 E] 的證明

範例：
- GPA 2.5 逆襲帝國理工，顧問做了什麼？
- 雅思卡關者的秘密武器：這份備考策略讓你少考一次
- 錄取多所學校，竟是選錯方向的證明？
```

## 8. CTA 模板

### 類型一｜低門檻留言型
```
留言「[關鍵字]」，[具體獲得物] 一次看！

範例：
- 留言「英國排名」，漲幅表格一次看！

- 留言「GPA」，解鎖低分逆襲案例懶人包！
```

```
立即預約 1 對 1 諮詢，顧問免費幫你[具體服務]！

範例：

```

## 9. 廣告結構

### 五步敘事公式（適用輪播圖卡）

```
第 1 張｜HOOK（封面）
→ 震撼大標 + 主張
→ 學生照片或錄取信畫面
→ 目標：讓人停下來滑

第 2 張｜PROBLEM（背景故事）
→ 描述學生原始條件
→ 點出面臨的困境與焦慮
→ 讓讀者產生「這說的就是我」的共鳴

第 3 張｜CHALLENGES（申請挑戰）
→ 具體列出 3 大難題（Bullets 版型）
→ 以數據或案例增加可信度
→ 讓問題「看得見」

第 4 張｜SOLUTION（顧問策略）
→ 針對性解決方案
→ 強調「身份重塑」或「賽道修正」
→ 展示 GoStudy 獨特方法論

第 5 張｜RESULT（錄取成果）
→ 秀出 Offer 信件截圖
→ 加入學生真實回饋引言
→ 結尾放 CTA
```

## 10. 社群貼文結構

### 結構 A｜話題型（適合趨勢資訊）

```
① 震撼開場（數據或反常識結論）
   → 「2026 QS 排名出爐，有所學校跌了 40 名！」

② 痛點共鳴（讀者最在乎什麼）
   → 「如果你還在用 3 年前的排名選校，等於白費力氣...」

③ 專家分析（GoStudy 洞察）
   → 列出 3–4 個關鍵發現，附數據來源

④ 選校建議（具體可操作）
   → 「根據以上，我們建議這類學生優先考慮 XX 類型學校」

⑤ CTA
   → 低門檻留言型 或 預約諮詢型
```

### 結構 B｜案例型（適合成功故事）

```
① 錄取喜報（結果先行）
   → 「🎓 恭喜！GPA 2.7 的 Chloe 拿到帝國理工 Offer！」

② 背景故事（學生條件說明）
   → 描述原始情況：科系、GPA、工作經驗、語言成績

③ 申請死穴（困難與挑戰）
   → 「她面臨的 3 大難題：XXX / XXX / XXX」

④ 策略拆解（GoStudy 如何介入）
   → 「顧問做了這 3 件事：身份重塑 / 選校調整 / SOP 故事化」

⑤ 顧問小叮嚀（專業洞察）
   → 1–2 句話的申請智慧，可引用顧問名言

⑥ CTA
   → 「你的背景也可以這樣操作，私訊 GOSTUDY 來聊！」
```

## 11. 禁止事項

### 內容禁忌

| 禁止行為 | 原因 | 替代做法 |
|---------|------|---------|
| 使用「套版文書」一詞 | 損害品牌差異化定位 | 改說「客製化申請策略」 |
| 給出模糊建議 | 削弱說服力 | 必須附上具體數據或案例 |
| 迴避 Study Gap / 低 GPA 話題 | 讓讀者感覺不被接納 | 主動提出並說明轉化策略 |
| 宣稱「保證錄取」 | 違反誠信原則 | 改說「最大化錄取機率」 |
| 過度謙遜的語氣 | 不符品牌自信定位 | 使用洞察型、挑戰型語氣 |
| 複製排名數字卻不附來源 | 降低公信力 | 必須標注 QS / THE / 教育部等來源 |

### 視覺禁忌

- ❌ 紫色漸層配色（AI 濫用配色）
- ❌ 大量純白背景（無品牌感）
- ❌ 字體混用超過 3 種
- ❌ 圖卡內文字超過 80 字
- ❌ Emoji 堆疊超過 3 個連用
