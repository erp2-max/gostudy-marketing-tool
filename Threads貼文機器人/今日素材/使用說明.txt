📌 今日素材資料夾 — 使用說明
════════════════════════════════════

【資料夾結構】

每天建立一個以「日期」命名的資料夾，格式：YYYY-MM-DD
例如：2026-04-13/

日期資料夾裡面再建兩個子資料夾：
  📁 第一篇/   ← 中午那篇的素材
  📁 第二篇/   ← 下午那篇的素材

════════════════════════════════════

【內容優先順序】

每一篇的判斷邏輯是：榜單 > 校代培訓 > 時事

也就是說：
  ✅ 資料夾裡有榜單素材（offer/CV/SOP）→ 自動產榜單文
  ✅ 資料夾裡有「校代」開頭的檔案      → 自動產校代文
  ✅ 資料夾裡有「時事」開頭的檔案      → 用你提供的素材產時事文
  ✅ 資料夾是空的                       → 自動搜尋 Threads 熱門話題產時事文

════════════════════════════════════

【各方向要丟哪些檔案？】

🏆 榜單（學生錄取分享）
  → offer.pdf 或 offer 截圖（.jpg/.png 也可以）
  → CV.pdf（學生履歷）
  → SOP.pdf（申請文件）
  → （選填）Writing Sample.pdf

🏫 校代培訓（學校亮點分享）
  → 校代資料.txt 或 校代資料.docx
     ⚠️ 檔名必須包含「校代」兩個字
     （把顧問提供的學校亮點資訊貼進去就好）

📰 時事（有素材）
  → 時事.txt 或 時事.docx
     ⚠️ 檔名必須包含「時事」兩個字
     （把你看到的話題、新聞連結、想法貼進去就好）

📰 時事（自動）
  → 不用放任何東西，資料夾留空即可
     → 排程會同時搜尋：
       ① Google Trends 台灣當天熱門話題
       ② Threads 公開頁面熱門貼文（無需登入）
       交叉比對後，從考試/新知/娛樂/盛事四個方向
       選最適合套留學角度的話題來產出草稿

════════════════════════════════════

【實際操作範例】

📅 今天有一個榜單、下午想自動找時事：
  2026-04-14/
    第一篇/
      offer.pdf、CV.pdf、SOP.pdf
    第二篇/              ← 空著，自動走時事

📅 今天有兩個榜單：
  2026-04-14/
    第一篇/
      offer.pdf、CV.pdf、SOP.pdf（學生A）
    第二篇/
      offer.pdf、CV.pdf、SOP.pdf（學生B）

📅 今天有榜單 + 你有想寫的時事話題：
  2026-04-14/
    第一篇/
      offer.pdf、CV.pdf、SOP.pdf
    第二篇/
      時事.txt    ← 把話題或想法寫進去，你來定主題

📅 今天沒有任何素材：
  → 不用建資料夾，排程兩篇都自動找時事話題

════════════════════════════════════

【排程時間】

🕙 每天 11:00 自動產出「第一篇」草稿
🕒 每天 15:00 自動產出「第二篇」草稿

→ 草稿會存到「草稿輸出」資料夾
→ 看完草稿，改一改就可以排程發文（12點~3點之間）

════════════════════════════════════

【想改寫作風格或 prompt？】

去「設定檔」資料夾直接修改對應的 .txt 檔：
  📄 Threads人設.txt        → 修改 Vivian 人設與寫作風格
  📄 offer分析prompt.txt    → 修改 offer 分析邏輯
  📄 文章風格範例.xlsx       → 新增/替換參考範例文章

改完存檔，下次排程自動套用新版本，不需要通知我。
